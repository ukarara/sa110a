# W3 git flow
增加團隊合作效率

## 軟體開發模式
1. 瀑布模型（Waterfall Model）
2. 螺旋模型（Spiral Model）
    * 演化模型(evolutionary model)
    * 快速原型模型（Rapid Prototype Model）
    * 增量模型（Incremental Model）

敏捷開發比較像 2 的螺旋模型，至於用《演化、快速原型或增量》則都有可能。(通常會採用增量的方式)
